# Java-Project-Checkers

This was a university project in a Java class; it is a recreation of the well-known game checkers. There were specific requirements for the project that I had to follow, like having interfaces, using serialization, and other stuff I had to bend the project around, so the design choices don't necessarily make complete sense, but the project still shows familiarity and capability of working with Java.

## Dependencies

- Java Development Kit (JDK) 17
- Maven

## Running

```bash
mvn javafx:run
```
